ZenLang on Solo5 unikernel base(sample)
===============================

Requirements
-----------
- zen on PATH
- wget
If you don't have own solo5 build, there are build_ script.  
```sh build__solo5.sh```

Build and Run
-------------

To build: ```zen build```  
To run: ```zen build run```
